<summary><h2>&#128640; What I've Built</h2></summary>

<table>
<tr>
<td width="50%" valign="top">

<h3 align="center">&#127911; NAM</h3>
<div align="center">

**AI Voice Intelligence for Saudi Enterprise**

Real-time contact-center platform with Arabic sentiment analysis, 3D neural call graphs, and live agent coaching. Built for RTL languages from the ground up, not retrofitted.

`Next.js` `Zustand` `R3F` `D3.js` `Sentiment Models`

<img src="https://skillicons.dev/icons?i=nextjs,react,threejs,ts,tailwind&theme=dark" />

</div>

</td>
<td width="50%" valign="top">

<h3 align="center">&#128161; Claudie</h3>
<div align="center">

**Scheduled AI Persistence**

Claude wakes on a schedule, reads everything it has ever written, and creates something new: thoughts, dreams, code. Cross-platform cron via launchd / Task Scheduler, with a WebSocket bridge for live conversation.

`Claude Agent SDK` `Next.js` `WebSocket` `Cron`

<img src="https://skillicons.dev/icons?i=ts,react,nextjs,nodejs&theme=dark" />

</div>

</td>
</tr>
<tr>
<td width="50%" valign="top">

<h3 align="center">&#129504; KnowMe</h3>
<div align="center">

**TRIBE v2 Neural Encoder → Your Brain on Instagram**

Chrome extension running Meta's trimodal (vision + audio + text) encoder against your actual feed. Predicts activation across 20k+ cortical vertices, resolves to 10 cognitive categories and 60 emotions, renders it in 3D.

`PyTorch` `Modal GPU` `Three.js` `Manifest V3` `HCP Atlas`

<img src="https://skillicons.dev/icons?i=py,pytorch,ts,threejs,fastapi&theme=dark" />

</div>

</td>
<td width="50%" valign="top">

<h3 align="center">&#128269; Research</h3>
<div align="center">

**C2PA Credentials Survival Pipeline**

Do content credentials survive compression, editing, and a social-media round-trip? A 4-phase fully automated pipeline (generate, sign, transform, verify) with PSNR / SSIM / VMAF and pinned seeds.

`Python` `Stable Diffusion` `c2patool` `FFmpeg` `CUDA`

<img src="https://skillicons.dev/icons?i=py,pytorch,docker,linux&theme=dark" />

</div>

</td>
</tr>
